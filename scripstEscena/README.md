# Narrative and Interactive Conversation System

## English

This module implements the game's dialogue interaction system, including cinematic scenes, narrative control, and video playback integration within the story.

---

## Conversation System

The dialogue system is based on sequential text progression, allowing the player to advance the story through:

- Keyboard input  
- Narrative flow control  
- Dynamic UI updates  

The system enables structuring the story into narrative blocks.

---

## Narrative Video Integration

A cinematic sequence playback system was implemented using `VideoPlayer`.

Main features:

- Video playback inside narrative sequences  
- Detection of video completion events  
- Temporary blocking of player input during playback  
- Automatic continuation of conversation after video ends  

Videos are used to reinforce important story moments.

---

## Character and Visual Narrative Management

The system controls:

- Character name display on screen  
- UI color modification depending on the speaker  
- Boss animation behavior during conversations  

A state control pattern is used to synchronize:

- Dialogue  
- Animation  
- Video playback  
- Scene flow  

---

## Contribution to the Project

This module demonstrates skills in:

- Interactive narrative system development  
- Synchronization between UI, animation, and video playback  
- Event-driven game flow control  
- Design of cinematic interactive experiences

## Español

# Sistema Narrativo y de Conversaciones Interactivas

Este módulo implementa el sistema de diálogos interactivos del juego, incluyendo escenas cinematográficas, control narrativo y reproducción de videos dentro de la historia.

---

## Sistema de Conversaciones

El sistema de diálogo se basa en progreso secuencial del texto, permitiendo al jugador avanzar la historia mediante:

- Entrada de teclado  
- Control del flujo narrativo  
- Actualización dinámica de la interfaz  

El sistema permite organizar la historia en bloques narrativos.

---

## Integración de Video Narrativo

Se implementó un sistema de reproducción de secuencias cinematográficas mediante `VideoPlayer`.

Características principales:

- Reproducción de video dentro de la narrativa  
- Detección del evento de finalización del video  
- Bloqueo temporal de la entrada del jugador durante la reproducción  
- Continuación automática de la conversación después del video  

Los videos se utilizan para reforzar momentos narrativos relevantes.

---

## Gestión Visual de Personajes y Narrativa

El sistema controla:

- Visualización del nombre del personaje en pantalla  
- Cambios de color en la interfaz según el hablante  
- Control de animaciones del jefe durante la conversación  

Se utiliza un modelo de control de estado para sincronizar:

- Diálogo  
- Animación  
- Reproducción de video  
- Flujo de escena  

---

## Qué aporta este sistema al proyecto

Este módulo demuestra habilidades en:

- Desarrollo de sistemas narrativos interactivos  
- Sincronización entre interfaz, animación y video  
- Control de flujo mediante eventos  
- Diseño de experiencias cinemáticas interactivas
